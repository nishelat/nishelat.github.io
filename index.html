Featured Projects
HackTube ● SBHacks 2017 ● 2nd Best Overall, Best Hack Against Harassment
~ Featured as a "Staff Pick" on Devpost ~

I wrote the JavaScript algorithms and designed the JSON file for this Google Chrome extension which fights online harassment by censoring derogatory YouTube comments. It dynamically accesses each page every quarter of a second and searches comments for specific words which are considered malicious, and then deletes the bad comments.

Succinct ● HackUCI 2017 ● Organizer's Choice Award

I led the backend members of my team and implemented Flickr's photosearch API using PHP for this web app which turns essays into visual presentations. It uses the Google Cloud Natural Language Processing API to find the main ideas of an essay, and then uses the Flickr API to find matching photos. It then composes a slideshow about the essay using both the photos and the extracted main ideas.
